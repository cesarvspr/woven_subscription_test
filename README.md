## Woven subscriptions test
Solution for a woven test, this is made by myself. 

### Requirements
	. "github.com/onsi/ginkgo"
	. "github.com/onsi/gomega"

To run the tests go inside the `woven_challenge_test` folder and run:

```bash
 go test -v
```
